### How to install
1. Open Sublime Text 3
2. Ctrl + Shift + P
3. Search "Add Repository"
4. Input "https://github.com/ErdemAkin/Titan-Edge-Highlighter"
5. Ctrl + Shift + P
6. Search "Install Package"
7. Search "Titan-Edge-Highlighter"
8. Enter ^^
